# /draft-incident-notice

Generates a ready-to-post initial incident notice in under 2 minutes.

## Required Input

Before drafting, confirm:
1. Protocol name
2. What is confirmed (not suspected)
3. What action has been taken (if any)
4. What users should do right now

## Output

Provides three versions simultaneously:
- X/Twitter (short, urgent)
- Discord announcement (full)
- Telegram (formatted)

## Usage

"Run /draft-incident-notice — we are [PROTOCOL], we confirmed unusual activity 15 minutes ago, we have paused deposits, users should not interact with the app."

The agent will generate all three versions immediately.
